Flags:
-o ----> optional --outf    outfile name for storing images. Either .png or .dpf. (Default none)
-x ----> optional --xpar    x parameter, default time)
-y ----> optional --ypar    y parameter, default tod)
-s ----> optional --sb      side band number 0-3 as a list (no spacings), default [1])
-d ----> optional --det     detector number 0-19 as a list (no spacings), default [0])
-n ----> optional --nu      frequency number 0-33 as a list (no spacings), default [16])
-f ----> optional --freq    low/high frequencies to find avg frequency, default none)
-t ----> optional --time    set specific time values when plotting tod over frequency (default [1])
-l ----> optional --low     ymin for plot, (default none)
-u ----> optional --up      ymax for plot, (default none)
-b ----> optional --bins    bin-type from numpy.histogram (default fd)
-p ----> optional --plot    graph or hist, (default graph)
-h ----> optional --help    help)
